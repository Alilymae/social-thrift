# 🌱 Social Thrift

**Thrift. Style. Repeat. ♻️**

Social Thrift is a modern, community-powered platform that blends **sustainable fashion, AI-driven styling, and creative upcycling** into one seamless digital ecosystem. Built for the next generation of conscious consumers, creators, and sellers, Social Thrift transforms how people buy, sell, and reimagine fashion.

---

## ✨ Overview

Social Thrift is more than just a marketplace — it’s a **social, creative, and sustainable platform** where users can:

* Discover unique thrifted fashion pieces
* Sell and monetize clothing effortlessly
* Style outfits using AI-powered tools
* Learn and share upcycling techniques
* Engage with a like-minded creative community

## 🧱 Tech Stack

```txt
Frontend:   React 19 + Vite
Backend:    Express.js
Database:   Firebase Firestore
Auth:       Firebase Authentication
AI:         Google Gemini APIs
```
